header:- The header element specifies a header for a document or section. The 2 headings below are in this header tag.

h1:- First heading tag which has largest font size and keeps on decreasing as we go on to see second, third nd so on tags.

h2:- This is second heading tag whose font is lesser than first heading tag.

mark:- This is a mark tag which highlights the content within it.

section:-

A section is a thematic grouping of content, typically with a heading.

article:-

The article element specifies independent, self-contained content.An article should make sense on its own, and it should be possible to read it independently from the rest of the web site.
footer:- The footer element specifies a footer for a document or section.A footer element should contain information about its containing element. A footer typically contains the author of the document, copyright information, links to terms of use, contact information, etc.
